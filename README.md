# Conceptos y Práctica de Programación

Este repositorio contiene prácticas, conceptos y proyectos desarrollados utilizando diversas tecnologías de desarrollo web. A continuación, se detalla la estructura del proyecto y las tecnologías aplicadas en cada sección:

## 📁 Estructura del Repositorio

- **`0_Fundamentos_programacion/`**
  - **Descripción:** Conceptos básicos y fundamentos iniciales de programación.

- **`JavaScript_Fundamentos/`**
  - **Tecnologías:** JavaScript
  - **Descripción:** Ejercicios, teoría y conceptos profundizados de JavaScript.

- **`proyectos_JS/`**
  - **Tecnologías:** HTML5, CSS3, JavaScript (Vanilla JS)
  - **Descripción:** Diferentes proyectos prácticos construidos con JavaScript puro (como cronómetro, aplicación del clima, paleta de colores RGB, lista de tareas, etc.).

- **`React/`**
  - **Tecnologías:** React.js, JavaScript (JSX), Node.js, Vite
  - **Descripción:** Proyectos y prácticas enfocadas en la creación de interfaces de usuario utilizando la biblioteca React.

- **`Reforzamiento_TS_JS/`**
  - **Tecnologías:** TypeScript, JavaScript, Node.js
  - **Descripción:** Material de refuerzo, configuración y práctica para la transición y uso de TypeScript.
